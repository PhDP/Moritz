Wagner2
=======
[![Build status](https://travis-ci.org/PhDP/wagner2.svg?branch=master)](https://travis-ci.org/PhDP/wagner2)

Model of speciation / ecological diversification in spatial networks.

Building
--------

  $ mkdir build && cd build
  $ cmake ..
  $ make
